# BookMySpot - (Event Booking Application) 

BookMySpot is a comprehensive event booking app and website designed specifically for college events. Whether you're organizing a tech fest, workshop, concert or conference, BookMySpot offers a simple, efficient way to book and manage your events.

# Components of the App

User App
1. Login 
2. Register 
3. Welcome Screen
4. Home page :-  which show all upcoming events in the college 
5. My Events page :-  which show all booked events by the student
6. Profile Page
7. Event Page :- which show all information about event example location date etc.
8. QR code :- after booking the event QR code will be generated which will be used by event organizer to verify if ticket is valid by scanning it using admin app.

Admin App
1. Login 
2. Register 
3. Welcome Screen
6. Profile Page
7. Scanner :- Generated QR code of student ticket is scanned to verify if ticket is valid.

Web App
1. Login 
2. Register 
3. Create Event :- Organizer can Create Event from this page add thumbail, start & end date etc.
4. Past Event :- which show all past events took place in the college 
5. Ongoing Event :- which show all Ongoing events in the college 
6. Upcoming Event :- which show all upcoming events in the college .

# Purpose

1. To provide students with a hassle-free experience of managing their event tickets.
2. To eliminate the possibility of losing or damaging physical tickets.
3. To offer a digital solution for students to easily access and store their event tickets.
4. To help event organizers track their ticket sales and manage the event attendance more efficiently.
5. To reduce the likelihood of fraudulent activities during the ticket-selling process.
6. To provide a secure and reliable platform for both event organizers and attendees.
7. To increase transparency in the ticket-selling process and avoid any discrepancies in tracking the money earned from events.
8. To create a centralized platform for event management and ticketing, making it easier for organizers to manage multiple events at once.
9. To provide event organizers with a platform to advertise their events and reach a wider audience.
10. To enhance the overall experience of attending college events and promote a culture of efficient event management.
